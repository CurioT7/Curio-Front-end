PostRepliesCom Component

A component to display a single post reply in the user's inbox.

Parameters:
Name Type Description
props Object

The props of the component.

Properties
Name Type Description
title string

The title of the post.
noComments number

The number of comments on the post.
sender string

The username of the sender.
timestamp string

The timestamp when the reply was sent.
linkedSubreddit string

The name of the linked subreddit, if any.
message string

The content of the reply message.
itemId string

The ID of the reply message.
downvotedcomments Array

Array of downvoted comments.
upvotedcomments Array

Array of upvoted comments.
onBlockConfirmed function

Function to be called when block confirmation is confirmed.
id string

The ID of the reply message.

Example
import React from 'react';
import PostRepliesCom from './Post_Replies_Com';

const MyComponent = () => {
  return (
    <PostRepliesCom
      title="Post Title"
      noComments={10}
      sender="example_user"
      timestamp="2 hours ago"
      linkedSubreddit="example_subreddit"
      message="This is a sample post reply message."
      itemId="123456789"
      downvotedcomments={[{ _id: 'comment1' }, { _id: 'comment2' }]}
      upvotedcomments={[{ _id: 'comment3' }, { _id: 'comment4' }]}
      onBlockConfirmed={() => console.log('Block confirmed')}
      id="987654321"
    />
  );
};

export default MyComponent;
