UserNameMentionsCom Component

A component to display individual username mentions in messages.

Parameters:
Name Type Description
props Object

The props of the component.

Properties
Name Type Description
title string

The title of the mentioned post.
noComments number

The number of comments on the mentioned post.
sender string

The username of the sender of the message.
timestamp Date

The timestamp of the message.
linkedSubreddit string

The name of the linked subreddit, if any.
message string

The message content.
itemId string

The ID of the mentioned item.
downvotedcomments Array

Array of downvoted comments.
upvotedcomments Array

Array of upvoted comments.
onBlockConfirmed function

Function to execute when block confirmation is received.
postId string

The ID of the mentioned post.
id string

The ID of the message.

Example
import React from 'react';
import UserNameMentionsCom from './UserNameMentionsCom';

const MyComponent = () => {
  return (
    <UserNameMentionsCom 
      title="Sample Title" 
      noComments={10} 
      sender="sampleUser" 
      timestamp={new Date()} 
      linkedSubreddit="sampleSubreddit" 
      message="Sample message content" 
      itemId="123456789" 
      downvotedcomments={[]} 
      upvotedcomments={[]} 
      onBlockConfirmed={() => console.log("Block confirmation received")} 
      postId="987654321" 
      id="abcdefghi" 
    />
  );
};

export default MyComponent;
